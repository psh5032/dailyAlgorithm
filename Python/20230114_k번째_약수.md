> 인프런 - 김태원 강사님의 '파이썬 알고리즘 문제풀이 입문' 문제 풀이

# 나의 풀이
```python
import sys
sys.stdin=open("input.txt","rt")

n, k = map(int, input().split())

b=[]
for i in range(1, n+1):
  if n%i==0:
    b.append(i)
```
- b리스트의 길이가 k보다 작으면 k번째 수가 존재하지 않으므로 k의 수를 기준으로 출력할 값을 나눈다.
```python
if len(b)<k:
    print(-1)
else:
  c=b.pop(k-1)  
  print(c)
```
